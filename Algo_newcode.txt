1. Création d'une liste chaînée de poids
Algorithme : CreerListePoids
Variable
  tete : Poids
  courant : Poids
  i : entier

Début
  tete ← NULL
  courant ← NULL
  Pour i allant de 0 à taille - 1 faire
    Créer Poids (nouveau)
    Affecter à nouveau.valeur : poids[i]
    Affecter à nouveau.suivant : NULL
    Si tete est NULL alors
      Affecter à tete : nouveau
    Sinon
      Affecter à courant.suivant : nouveau
    Fin si
    Affecter à courant : nouveau
  Fin pour
  Retourner tete
Fin


2. Initialisation d'un neurone
Algorithme : InitNeur
Variable
  neurone : Neurone

Début
  Créer Neurone (neurone)
  Affecter à neurone.listePoids : CreerListePoids(poids, nombreEntrees)
  Affecter à neurone.biais : biais
  Affecter à neurone.suivant : NULL
  Retourner neurone
Fin








3. Calcul de la sortie d'un neurone
Algorithme : OutNeurone
Variable
  somme : entier
  courant : Poids
  i : entier

Début
  somme ← 0
  Affecter à courant : neurone.listePoids
  Pour i allant de 0 à taille faire
    somme ← somme + courant.valeur * entrees[i]
    Affecter à courant : courant.suivant
  Fin pour
  Si somme ≥ neurone.biais alors
    Retourner 1
  Sinon
    Retourner 0
  Fin si
Fin


4. Algorithme pour InitCouche
Variables : 
couche : Couche (pointeur)
dernier : Neurone (pointeur)
nouveauNeurone : Neurone (pointeur)
i : entier

Début
Allouer mémoire pour couche
Affecter à couche.listeNeurones : NULL
Affecter à dernier : NULL
Pour i allant de 0 à nbNeurones - 1 faire :
Appeler InitNeur avec poids[i], nbEntrees, biais[i] et affecter le résultat à nouveauNeurone
Si couche.listeNeurones est NULL alors :
Affecter nouveauNeurone à couche.listeNeurones
Sinon :
Affecter nouveauNeurone à dernier.suivant
Affecter nouveauNeurone à dernier
Retourner couche
Fin


5.Algorithme pour OutCouche

Variables:
sorties : tableau d’entiers
courant : Neurone (pointeur)
i : entier

Début:
Affecter à courant : couche.listeNeurones
Affecter à i : 0
Tant que courant n’est pas NULL faire :
Appeler OutNeurone avec courant, entrees, taille et stocker le résultat dans sorties[i]
Affecter à courant : courant.suivant
Incrémenter i
Retourner sorties
Fin
